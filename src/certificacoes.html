ISO 9001

A certificação ISO 9001, que é uma norma internacionalmente reconhecida que certifica o Sistema de Gestão da Qualidade (SGQ) e define os requisitos (ferramentas de padronização) para a implantação do sistema em uma organização. Um dos requisitos para que a empresa de construção civil tenha a certificação ISO 9001 é ter laudo técnico de todos os materiais utilizados na obra.

NBRs

As principais NBRs exigidas em uma obra são:

NBR 15575

A NBR 15575 é uma Norma de Desempenho de Edificações Habitacionais criada com o objetivo de assegurar que obras de imóveis residenciais atendam a um critério mínimo de qualidade e segurança. Sendo assim as construtoras precisam se adequar ao alto padão de qualidade exigido além de demonstrar através de análises que as determinações estão sendo respeitadas. Por se tratar de uma norma complexa ela é dividida em seis requisitos principais: requisitos gerais, requisitos para os sistemas estruturais, requisitos para o sistemas de pisos, requisitos para os sistemas de vedações verticais, requisitos para os sistemas de coberturas e requisitos para os sistemas hidrossanitários.

Nome Aplicação
NBR 12655 Esta norma é aplicável a concreto de cimento Portland para estruturas moldadas na obra, estruturas pré-moldadas e componentes estruturais pré-fabricados para edificações e estruturas de engenharia.
NBR 7211 Esta norma especifica os requisitos exigíveis para recepção e produção dos agregados miúdos e graúdos destinados à produção de concretos de cimento Portland.
NBR NM 248 Esta norma prescreve o método para a determinação da composição granulométrica de agregados miúdos e graúdos para concreto.
NBR 5739 Esta norma prescreve um método de ensaio pelo qual devem ser ensaiados à compressão os corpos-de-prova cilíndricos de concreto.
NBR 7222 Esta norma prescreve o método para determinação da resistência à tração por compressão diametral de corpos de prova e testemunhos cilíndricos de concreto e argamassa.
NBR 8522 Esta norma estabelece os métodos de ensaio para a determinação dos módulos estáticos de elasticidade e de deformação à compressão do concreto endurecido, em corpos de prova cilíndricos.

Ensaios

Segue abaixo uma relação dos ensaios que podem ser exigidos das Construtoras para atenderem as NBRs citadas acima ou outras exigidas pela ABNT:

  • Análise química
  • Determinação da estabilidade
  • Determinação da plasticidade
  • Determinação da finura
  • Determinação de retenção de água
  • Ensaio a compressão simples
